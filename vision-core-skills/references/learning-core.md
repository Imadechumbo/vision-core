# LEARNING CORE Doctrine

VISION CORE does not learn from arbitrary text.

Validated learning requires:
1. Sanitization
2. Reproduction
3. Benchmark or mission execution
4. Aegis OK
5. PASS GOLD true
6. Auditable storage
7. Regression guard when possible

Lifecycle:
candidate → triaged → reproduced → benchmarked → solved_PASS_GOLD → validated_memory → regression_guard
